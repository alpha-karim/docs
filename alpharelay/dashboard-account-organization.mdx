The AlphaRelay web app centers on the mission dashboard plus focused Mission Overwatch consoles per flight. Start from **Mission Dashboard** after sign-in, then open Mission Overwatch with **Open in console** whenever you need logging, uploads, playback, reporting, exports, or the live relay surface for active ops.

## Mission dashboard

Open **Mission Dashboard** from the web app after signing in.

Use the dashboard to:

- Open Mission Overwatch for a mission (**Open in console**)
- Review active and closed missions
- Search missions, events, footage, screenshots, and metadata
- Filter search by missions, events, footage, screenshots, active status, or quick events
- Review mission totals, storage usage, event cadence, event sources, mission throughput, and recent timeline activity
- Open closed mission reports
- Generate or edit an Alpha AI executive summary for a closed mission
- Print the after-action report

For submitted reports, the dashboard also shows supervisor review queues when your role can act on them.

## Account settings

Open **Account** from the dashboard navigation.

Use account settings to:

- Change your password
- Request an email change
- Save your full name
- Save your rank or title

Your name and rank can appear in organization member lists, mission attribution, and custody logs. If they are blank, AlphaRelay falls back to your account email where needed.

## Organization page

Organization owners and admins can open **Organization** from the dashboard navigation.

Use the organization page to:

- Review indexed storage for mission footage, screenshots, and mission photos
- Add or update team members
- Change member roles
- Set membership status to active, invited, or disabled
- Remove memberships without deleting the underlying AlphaRelay account
- Save member names and ranks
- Set the organization time zone used across the dashboard and focused Mission Overwatch sessions
- View pilot tablet/phone license keys and seat usage
- Copy the pilot license key or use the QR code from the license row

<Note>
  If you cannot open the organization page, you may not be an owner or admin for the selected organization.
</Note>

## Pilot tablet/phone licenses

The organization page can show active pilot license keys and QR codes that open the Android pilot app through an `alpharelay://license?key=...` link.

Use this page when a pilot needs to activate a tablet or phone that already has the Android pilot app installed. If a key needs to be created, revoked, reactivated, or expanded, contact your AlphaRelay platform team.

## Organization time zone

The organization time zone controls how mission times appear across the dashboard and Mission Overwatch timelines. Choose the zone that matches the team's operational reporting standard.

If the organization keeps the default setting, AlphaRelay uses the browser or persisted local time behavior available to the current session.
