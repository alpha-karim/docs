## Network and hardware

Before starting a mission:

- Put the pilot tablet and command device on the same Wi-Fi network.
- Connect the pilot tablet to the drone controller.
- Power on the drone controller.
- Power on the aircraft.

The command device can be a laptop, desktop, another tablet or phone, or the same device running the app.

## Start the relay and mission console

**On the pilot tablet**

1. Tap **Start Relay + DJI**.
2. Complete any license step if prompted.

**In the web app**

1. From the home or command dashboard, click **Open Mission Console**.
2. Under **Mission**, enter the mission name.
3. Select a **Scenario Template**, if your team uses one.
4. Click **Start Mission**.
5. Under **Live Overwatch Screen**, enter the IPv4 address shown on the pilot tablet (for example, `192.168.1.224`).
6. Click **Connect**.

<Tip>
  You can also leave the IP field blank and use **Connect** or **Scan LAN**, depending on the on-screen hint.
</Tip>

If the browser asks you to trust the pilot relay certificate:

1. Click **Trust Relay Certificate**.
2. Complete the trust page in the new tab.
3. Return to Mission Overwatch.
4. Connect again.
