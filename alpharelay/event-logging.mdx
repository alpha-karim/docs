## Logging events on the pilot tablet

On the pilot tablet, operators can log events in several ways:

- Use the quick event controls.
- Say “relay” when voice logging is enabled.
- Type a custom event line in the pilot control panel.

Quick event presets can match what Mission Overwatch displays for the selected scenario.

## Logging events in Mission Overwatch

In the web console, use either **Event Logging Quick Buttons** or **Manual Event** → **Log Manual Event**.

## What gets stored

Logged events are:

- Timestamped
- Added to the **Event Log**
- Associated with mission activity
- Linked to screenshots from the pilot view when the relay is connected

If the system is offline or delayed, events and screenshots may sync once connectivity returns.
