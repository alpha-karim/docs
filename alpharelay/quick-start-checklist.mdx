Use this checklist for a first training run or a quick pre-mission rehearsal.

## Before you start

- Sign in at [https://www.alpha-relay.com](https://www.alpha-relay.com).
- Install and open the AlphaRelay pilot app on the Android pilot tablet.
- Activate the pilot license if prompted.
- Connect the pilot tablet to the drone controller.
- Power on the controller and aircraft.
- Put the pilot tablet and Mission Overwatch device on the same local Wi-Fi network.

## Start a test mission

1. On the pilot tablet, tap **Start Relay + DJI**.
2. In the web app, open **Mission Console**.
3. Enter a mission name such as `Training mission`.
4. Select a scenario template if your team uses one.
5. Click **Start Mission**.
6. Enter the IPv4 address shown on the pilot tablet, or use **Scan LAN** when available.
7. Click **Connect**.
8. Confirm the live pilot feed appears in Mission Overwatch.

## Prove the workflow

1. Log one quick event from the pilot tablet.
2. Log one event from Mission Overwatch.
3. Confirm both entries appear in the **Event Log**.
4. Capture a short flight or short test recording.
5. Land the aircraft using normal DJI safety procedures.
6. Click **Close Mission** in Mission Overwatch.

## Review the mission

1. Upload or wait for the mission footage.
2. Open **Play with Events**.
3. Confirm event markers appear on the timeline.
4. Use **Tag event** to add one post-mission event from a video frame.
5. Generate the after-action report.
6. Export an offline package.

When this checklist passes, the team has validated the core AlphaRelay path: live relay, structured events, footage review, post-mission tagging, report generation, and offline export.
